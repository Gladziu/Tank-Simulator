## Tank Simulator
Simple university project made in opengl and cpp.


## Setup
To run this project, open the .sln file.

 
## Controls
`W` `A` `S` `D` - movement

`Q` `E` - turret roate left and right

`R` `F` - gun rotate up and down

`SPACE` - shot


&#8592; `/\` `\/` `>` - camera movement

`Page Up` `Page Down` - zoom in, zoom out
